# 🏠 House Price Prediction System

An AI-powered Machine Learning web application that predicts house prices using multiple regression models and a Streamlit interface.

---

## 🚀 Features

- Data preprocessing and cleaning
- Feature engineering
- Multiple ML models (Random Forest, XGBoost, Ridge, Lasso)
- Model comparison
- Interactive Streamlit UI
- Prediction history tracking
- Download prediction reports
- Auto-generated charts and visualizations

---

## 🧠 Machine Learning Models

- Random Forest Regressor
- XGBoost Regressor
- Ridge Regression
- Lasso Regression

---

## 🛠️ Tech Stack

Python, Pandas, NumPy, Scikit-learn, XGBoost, Streamlit, Matplotlib, Seaborn, Joblib

---

## ▶️ How to Run

```bash
pip install -r requirements.txt

```bash
python train.py

```bash
streamlit run app.py
